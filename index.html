Projeto Zelda

Um Projeto de homenagem a franquia The Legend of Zelda, contando de maneira simples as principais sagas da franquia, linha temporal dos jogos e UI organizada e responsiva.

  • Interface organizada e responsiva;
  • Divisão dos jogos da saga de maneira temporal.

GITFIND

Um buscador que consulta a API do GitHub via metódo Fetch, exibindo o usuário e todos os seus repositórios.

  • Consulta via requisição;

Pokémon XS

Projeto Pokémon XS apresenta uma nova franquia Pokémon ambientada em São Paulo, com 200 novos Pokémon inspirados na cultura brasileira. O objetivo é se inspirar na franquia Pokémon para simular uma arquitetura de um site de jogos, oferecendo atualizações do jogo, artes dos Pokémon e uma versão Beta do jogo.

  • Áudios funcionais de cada um dos Pokémon;
  • Cadastro e login de usuários;
  • Função de likes e favoritos com insights.

Sobre mim

Olá, sou Cauã Ribeiro, um entusiasta apaixonado por design e tecnologia, dedicado a trazer soluções inovadoras para a vida das pessoas através da programação e do design de interfaces.
